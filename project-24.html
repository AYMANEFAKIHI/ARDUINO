Project #24: Radar Sweep (Serial Plotter)

Build a scanning radar! A servo sweeps the ultrasonic sensor back and forth while you visualize the data using Arduino's Serial Plotter or Processing.

1. Concept and Description

In this project, we will explore the following concepts:

  • Combining servo motors with sensors
  • Using the Serial Plotter for visualization
  • Creating scanning/sweeping mechanisms
  • Data formatting for graphing

2. Components Needed

  • (1) Arduino Uno
  • (1) HC-SR04 Ultrasonic
  • (1) SG90 Servo Motor
  • (1) Breadboard
  • (8+) Jumper Wires

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

/*
 * Project #24: Radar Sweep (Serial Plotter)
 * The Arduino Student Lab
 * 
 * Servo sweeps ultrasonic sensor 180 degrees
 * Data output formatted for Serial Plotter
 */

#include <Servo.h>

// ============================================
// PIN DEFINITIONS
// ============================================
const int TRIG_PIN = 9;
const int ECHO_PIN = 10;
const int SERVO_PIN = 6;

// ============================================
// CONFIGURATION
// ============================================
const int SWEEP_MIN = 15;     // Minimum angle (degrees)
const int SWEEP_MAX = 165;    // Maximum angle (degrees)
const int SWEEP_STEP = 1;     // Degrees per step
const int SWEEP_DELAY = 30;   // Delay between steps (ms)

// ============================================
// OBJECTS
// ============================================
Servo radarServo;

// ============================================
// SETUP
// ============================================
void setup() {
  pinMode(TRIG_PIN, OUTPUT);
  pinMode(ECHO_PIN, INPUT);
  
  radarServo.attach(SERVO_PIN);
  radarServo.write(90);  // Start at center
  
  Serial.begin(9600);
  
  // Wait for servo to reach position
  delay(1000);
}

// ============================================
// MAIN LOOP
// ============================================
void loop() {
  // Sweep from left to right
  for (int angle = SWEEP_MIN; angle <= SWEEP_MAX; angle += SWEEP_STEP) {
    scanAtAngle(angle);
  }
  
  // Sweep from right to left
  for (int angle = SWEEP_MAX; angle >= SWEEP_MIN; angle -= SWEEP_STEP) {
    scanAtAngle(angle);
  }
}

// ============================================
// HELPER FUNCTIONS
// ============================================

void scanAtAngle(int angle) {
  // Move servo to angle
  radarServo.write(angle);
  delay(SWEEP_DELAY);
  
  // Measure distance
  float distance = measureDistance();
  
  // Clamp distance for display (0-200cm range)
  if (distance > 200) distance = 200;
  if (distance < 0) distance = 0;
  
  // Output format for Serial Plotter
  // Format: angle,distance
  Serial.print(angle);
  Serial.print(",");
  Serial.println(distance);
}

float measureDistance() {
  // Send trigger pulse
  digitalWrite(TRIG_PIN, LOW);
  delayMicroseconds(2);
  digitalWrite(TRIG_PIN, HIGH);
  delayMicroseconds(10);
  digitalWrite(TRIG_PIN, LOW);
  
  // Measure echo
  long duration = pulseIn(ECHO_PIN, HIGH, 30000);
  
  // Calculate distance
  float distance = duration * 0.0343 / 2.0;
  
  // Handle no-echo
  if (duration == 0) distance = 200;
  
  return distance;
}

/*
 * USING THE SERIAL PLOTTER
 * 
 * 1. Upload this code
 * 2. Go to Tools ‚Üí Serial Plotter (or Ctrl+Shift+L)
 * 3. Set baud rate to 9600
 * 4. Watch the sweep visualization!
 * 
 * The plotter shows two lines:
 * - Blue line = angle (0-180, sawtooth wave)
 * - Red line = distance (varies with obstacles)
 * 
 * FOR A REAL RADAR DISPLAY:
 * Use Processing (processing.org) with this data
 * to create a proper radar sweep visualization!
 *

 * https://github.com/arduino-student-lab/radar
