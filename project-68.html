Project #68: MPU-6050 Accelerometer/Gyro

Read raw motion data (acceleration and rotation) from the MPU-6050 sensor.

1. Concept and Description

The MPU-6050 is a 6-axis MotionTracking device that combines a 3-axis gyroscope and a 3-axis accelerometer.

2. Components Needed

  • Arduino Uno
  • MPU-6050 Module

3. Physical Montage (Wiring Instructions)

VCC to 5V, GND to GND, SDA to A4, SCL to A5.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.


#include 
const int MPU_addr=0x68;
int16_t AcX,AcY,AcZ,Tmp,GyX,GyY,GyZ;

void setup(){
  Wire.begin();
  Wire.beginTransmission(MPU_addr);
  Wire.write(0x6B); 
  Wire.write(0); 
  Wire.endTransmission(true);
  Serial.begin(9600);
}

void loop(){
  Wire.beginTransmission(MPU_addr);
  Wire.write(0x3B); 
  Wire.endTransmission(false);
  Wire.requestFrom(MPU_addr,14,true);
  AcX=Wire.read()<<8|Wire.read();
  AcY=Wire.read()<<8|Wire.read();
  AcZ=Wire.read()<<8|Wire.read();
  // ... read others ...
  Serial.print("AcX = "); Serial.println(AcX);
  delay(333);
}
