Project #33: LCD Custom Characters & Animation

Go beyond text! Learn how to design and display your own custom icons, symbols, and simple animations on the LCD screen.

1. Concept and Description

The LCD controller (HD44780) has a built-in character set, but it also allows you to define up to 8 custom characters (0-7). Each character is a 5x8 pixel grid. By defining which pixels are on or off, we can create battery icons, smileys, or even simple animations.

  • Understanding the 5x8 pixel grid
  • Using binary to design characters
  • createChar() function
  • Animating by switching characters

2. Components Needed

  • (1) Arduino Uno
  • (1) 16x2 LCD with I2C Backpack
  • (4) Jumper Wires

3. Physical Montage (Wiring Instructions)

Standard I2C wiring:

  1. VCC to 5V
  2. GND to GND
  3. SDA to A4
  4. SCL to A5

4. The Code (Arduino Sketch)

This code creates a "Pac-Man" style animation and a battery charging icon.

/*
 * Project #33: LCD Custom Characters
 * The Arduino Student Lab
 * 
 * Creates custom icons (Pacman, Ghost, Battery)
 * and animates them.
 */

#include <Wire.h>
#include <LiquidCrystal_I2C.h>

LiquidCrystal_I2C lcd(0x27, 16, 2);

// Define Custom Characters (5x8 grid)
// 1 = Pixel ON, 0 = Pixel OFF

byte pacmanOpen[8] = {
  0b01110,
  0b11101,
  0b11111,
  0b11100,
  0b11111,
  0b11111,
  0b01110,
  0b00000
};

byte pacmanClosed[8] = {
  0b01110,
  0b11111,
  0b11111,
  0b11111,
  0b11111,
  0b11111,
  0b01110,
  0b00000
};

byte ghost[8] = {
  0b01110,
  0b11111,
  0b10101,
  0b11111,
  0b11111,
  0b11111,
  0b10101,
  0b00000
};

void setup() {
  lcd.init();
  lcd.backlight();
  
  // Register custom characters (0-7 allowed)
  lcd.createChar(0, pacmanOpen);
  lcd.createChar(1, pacmanClosed);
  lcd.createChar(2, ghost);
  
  lcd.setCursor(0, 0);
  lcd.print("Custom Chars!");
}

void loop() {
  // Animation Loop
  for (int i = 0; i < 16; i++) {
    lcd.setCursor(i, 1);
    
    // Frame 1: Open Mouth
    lcd.write(0); 
    delay(200);
    
    // Frame 2: Closed Mouth
    lcd.setCursor(i, 1);
    lcd.write(1);
    delay(200);
    
    // Clear the character behind (unless it's the start)
    lcd.setCursor(i, 1);
    lcd.print(" ");
  }
  
  delay(1000);
}

5. Explanation & Challenges

Binary Art: We define characters using arrays of bytes. 0b01110 represents one row of 5 pixels (0=off, 1=on). You can draw your own on graph paper first!

Challenges:

  • Battery Indicator: Create 4 custom characters representing 25%, 50%, 75%, and 100% battery and animate them charging.
  • Dino Run: Recreate the Chrome Dino game character.
  • Big Numbers: Use 4 custom characters combined to create giant digits that span 2 rows.
