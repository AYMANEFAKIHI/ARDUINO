Project #91: Timer Interrupts (Multitasking)

Run code at precise intervals without using delay().

1. Concept and Description

Interrupts allow the Arduino to do two things at once. We'll blink an LED while reading a sensor.

2. Components Needed

  • Arduino Uno
  • LED

3. Physical Montage (Wiring Instructions)

LED to D13.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

// Use TimerOne library or register manipulation
void setup() {
  // Configure Timer1 for 1Hz interrupt
}
ISR(TIMER1_COMPA_vect) {
  digitalWrite(13, !digitalRead(13));
}
