Project #96: Smart Plant Watering System

Automatically water your plant when the soil gets dry.

1. Concept and Description

Combines soil moisture sensing with a water pump.

2. Components Needed

  • Arduino Uno
  • Soil Sensor
  • Relay Module
  • Water Pump

3. Physical Montage (Wiring Instructions)

Sensor to A0. Relay to D7.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

if (analogRead(A0) > THRESHOLD) {
  digitalWrite(PUMP_PIN, HIGH);
  delay(2000); // Water for 2 seconds
  digitalWrite(PUMP_PIN, LOW);
}
