Project #56: Clap Switch (Sound-Activated)

Turn a device on or off by clapping your hands.

1. Concept and Description

Using the sound sensor, we can detect the sharp spike of a clap and toggle a relay or LED.

2. Components Needed

  • Arduino Uno
  • Sound Sensor
  • Relay Module (or LED)

3. Physical Montage (Wiring Instructions)

Sensor DO (Digital Out) to D2. Relay/LED to D13.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

const int sensorPin = 2;
const int relayPin = 13;
boolean state = false;

void setup() {
  pinMode(sensorPin, INPUT);
  pinMode(relayPin, OUTPUT);
}

void loop() {
  if (digitalRead(sensorPin) == HIGH) {
    state = !state;
    digitalWrite(relayPin, state ? HIGH : LOW);
    delay(500); // Debounce to prevent multiple toggles
  }
}
