Project #59: 4x4 Keypad Password Lock

Create a security system that requires a passcode to unlock.

1. Concept and Description

Matrix keypads allow you to input numbers. We'll use a 4x4 keypad to enter a password.

2. Components Needed

  • Arduino Uno
  • 4x4 Membrane Keypad
  • Servo Motor (Lock)
  • LEDs (Red/Green)

3. Physical Montage (Wiring Instructions)

Keypad Rows to D9-D6, Cols to D5-D2.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

#include 

const byte ROWS = 4; 
const byte COLS = 4; 
char keys[ROWS][COLS] = {
  {'1','2','3','A'},
  {'4','5','6','B'},
  {'7','8','9','C'},
  {'*','0','#','D'}
};
byte rowPins[ROWS] = {9, 8, 7, 6}; 
byte colPins[COLS] = {5, 4, 3, 2}; 

Keypad keypad = Keypad( makeKeymap(keys), rowPins, colPins, ROWS, COLS );
char password[] = "1234";
String input = "";

void setup(){
  Serial.begin(9600);
}

void loop(){
  char key = keypad.getKey();
  
  if (key){
    Serial.println(key);
    if(key == '#') {
      if(input == password) {
        Serial.println("Unlocked!");
      } else {
        Serial.println("Wrong Password");
      }
      input = "";
    } else {
      input += key;
    }
  }
}
