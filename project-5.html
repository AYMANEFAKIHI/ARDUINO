Project #5: SOS Morse Code Flasher

Create a classic SOS distress signal using Morse code! This project teaches pattern timing and introduces you to functions that make complex sequences manageable.

1. Concept and Description

In this project, we will explore the following concepts:

  • Understanding Morse code timing standards
  • Creating reusable functions for patterns
  • Organizing complex timing sequences
  • Using constants for easy customization

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

/*
 * Project #5: SOS Morse Code Flasher
 * The Arduino Student Lab
 * 
 * Flashes the international distress signal SOS
 * in proper Morse code timing
 */

// ============================================
// PIN DEFINITIONS
// ============================================
const int LED_PIN = 13;

// ============================================
// MORSE CODE TIMING (in milliseconds)
// Standard ratio: dot=1, dash=3, gaps follow same pattern
// ============================================
const int UNIT_TIME = 200;                    // Base unit: 200ms
const int DOT_DURATION = UNIT_TIME;           // 1 unit
const int DASH_DURATION = UNIT_TIME * 3;      // 3 units
const int INTRA_GAP = UNIT_TIME;              // Gap within a letter (1 unit)
const int LETTER_GAP = UNIT_TIME * 3;         // Gap between letters (3 units)
const int WORD_GAP = UNIT_TIME * 7;           // Gap between words/repeat (7 units)

// ============================================
// SETUP - Runs once at startup
// ============================================
void setup() {
  pinMode(LED_PIN, OUTPUT);
  
  Serial.begin(9600);
  Serial.println("SOS Morse Code Flasher");
  Serial.println("... --- ...");
}

// ============================================
// MAIN LOOP - Runs repeatedly
// ============================================
void loop() {
  // Flash SOS: ... --- ...
  
  Serial.println("Sending SOS...");
  
  // S = dot dot dot (...)
  flashS();
  delay(LETTER_GAP);
  
  // O = dash dash dash (---)
  flashO();
  delay(LETTER_GAP);
  
  // S = dot dot dot (...)
  flashS();
  
  // Pause before repeating
  Serial.println("Waiting to repeat...\n");
  delay(WORD_GAP);
}

// ============================================
// LETTER FUNCTIONS
// ============================================

// Flash the letter S: ...
void flashS() {
  Serial.print("S: ");
  dot();
  dot();
  dot();
  Serial.println();
}

// Flash the letter O: ---
void flashO() {
  Serial.print("O: ");
  dash();
  dash();
  dash();
  Serial.println();
}

// ============================================
// BASIC MORSE ELEMENTS
// ============================================

// Flash a dot (short)
void dot() {
  Serial.print(".");
  digitalWrite(LED_PIN, HIGH);
  delay(DOT_DURATION);
  digitalWrite(LED_PIN, LOW);
  delay(INTRA_GAP);
}

// Flash a dash (long)
void dash() {
  Serial.print("-");
  digitalWrite(LED_PIN, HIGH);
  delay(DASH_DURATION);
  digitalWrite(LED_PIN, LOW);
  delay(INTRA_GAP);
}

/*
 * MORSE CODE ALPHABET (for future expansion):
 * 
 * A: .-      N: -.
 * B: -...    O: ---
 * C: -.-.    P: .--.
 * D: -..     Q: --.-
 * E: .       R: .-.
 * F: ..-.    S: ...
 * G: --.     T: -
 * H: ....    U: ..-
 * I: ..      V: ...-
 * J: .---    W: .--
 * K: -.-     X: -..-
 * L: .-..    Y: -.--
 * M: --      Z: --..
 */

5. Explanation & Challenges

üîç How It Works

  1. Unit Time: All Morse timing is based on a single "unit" - here 200ms.
  2. dot() and dash(): These basic functions handle LED on/off with correct timing.
  3. flashS() and flashO(): Combine basic elements into complete letters.
  4. Main Loop: Calls letter functions with appropriate gaps between them.
  5. Serial Output: Shows what's being transmitted for debugging.

üöÄ Challenges

  • Full Alphabet: Create functions for all 26 letters using the reference table.
  • Name Flasher: Flash your name or initials in Morse code.
  • Add Sound: Use a piezo buzzer to beep along with the flashes.
  • Serial Input: Type a message in Serial Monitor and flash it in Morse.
  • Speed Control: Use a potentiometer to adjust the UNIT_TIME.
