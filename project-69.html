Project #69: MPU-6050 Tilt Angle Display

Calculate and display the tilt angle (pitch and roll) of the sensor.

1. Concept and Description

Using math (trigonometry), we can convert raw accelerometer values into angles.

2. Components Needed

  • Arduino Uno
  • MPU-6050

3. Physical Montage (Wiring Instructions)

Same as Project 68.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

// Simplified logic
// Calculate Pitch and Roll from AcX, AcY, AcZ
// pitch = atan(AcX / sqrt(AcY*AcY + AcZ*AcZ)) * 180/PI;
// roll = atan(AcY / sqrt(AcX*AcX + AcZ*AcZ)) * 180/PI;
