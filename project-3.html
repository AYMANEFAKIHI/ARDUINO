Project #3: Knight Rider LED Scanner

Create the iconic KITT scanner effect from Knight Rider! This project introduces arrays and for loops - essential programming concepts for controlling multiple components efficiently.

1. Concept and Description

In this project, we will explore the following concepts:

  • Using arrays to store multiple pin numbers
  • For loops to iterate through LEDs
  • Creating smooth scanning animations
  • Bidirectional sequences (forward and backward)

2. Components Needed

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

/*
 * Project #3: Knight Rider LED Scanner
 * The Arduino Student Lab
 * 
 * Creates a back-and-forth scanning LED effect
 * using arrays and for loops
 */

// ============================================
// CONFIGURATION
// ============================================
const int NUM_LEDS = 6;              // Total number of LEDs
const int SCAN_SPEED = 80;           // Delay between LED changes (ms)

// Array storing all LED pin numbers
// This makes it easy to loop through them
int ledPins[NUM_LEDS] = {2, 3, 4, 5, 6, 7};

// ============================================
// SETUP - Runs once at startup
// ============================================
void setup() {
  // Use a for loop to set all LED pins as outputs
  // Much cleaner than writing 6 separate pinMode() calls!
  for (int i = 0; i < NUM_LEDS; i++) {
    pinMode(ledPins[i], OUTPUT);
  }
}

// ============================================
// MAIN LOOP - Runs repeatedly
// ============================================
void loop() {
  // Scan from left to right (0 to NUM_LEDS-1)
  for (int i = 0; i < NUM_LEDS; i++) {
    lightOneLED(i);           // Turn on current LED
    delay(SCAN_SPEED);        // Wait
  }
  
  // Scan from right to left (NUM_LEDS-2 down to 1)
  // We skip the endpoints to avoid double-lighting
  for (int i = NUM_LEDS - 2; i > 0; i--) {
    lightOneLED(i);           // Turn on current LED
    delay(SCAN_SPEED);        // Wait
  }
}

// ============================================
// HELPER FUNCTIONS
// ============================================

/*
 * lightOneLED(index)
 * Turns ON only the LED at the specified index
 * Turns OFF all other LEDs
 * 
 * Parameters:
 *   index - which LED to light (0 to NUM_LEDS-1)
 */
void lightOneLED(int index) {
  // Loop through all LEDs
  for (int i = 0; i < NUM_LEDS; i++) {
    if (i == index) {
      digitalWrite(ledPins[i], HIGH);  // Target LED ON
    } else {
      digitalWrite(ledPins[i], LOW);   // All others OFF
    }
  }
}

5. Explanation & Challenges

üîç How It Works

  1. Arrays: ledPins[] stores all 6 pin numbers in order, so we can access them by index (0-5).
  2. For Loop in setup(): Instead of 6 pinMode() calls, one loop configures all pins.
  3. Forward Scan: Loop from index 0 to 5, lighting each LED in sequence.
  4. Backward Scan: Loop from index 4 down to 1 (skipping endpoints to avoid flickering at edges).
  5. lightOneLED(): This function ensures only ONE LED is lit at a time by turning all others off.

üöÄ Challenges

  • Speed Control: Add a potentiometer to control the scan speed in real-time.
  • Trailing Effect: Instead of turning off previous LEDs instantly, fade them using PWM.
  • Cylon Mode: Add a second row of LEDs that scans in the opposite direction.
  • More LEDs: Expand to 8 or 10 LEDs - just add to the array!
