Project #78: ESP8266 Wi-Fi Connection Basics

Connect your Arduino project to the internet using the ESP8266 module.

1. Concept and Description

The ESP8266 is a low-cost Wi-Fi chip. We can use it with AT commands or program it directly.

2. Components Needed

  • Arduino Uno
  • ESP-01 Module

3. Physical Montage (Wiring Instructions)

TX to D2, RX to D3 (via divider). VCC to 3.3V.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

#include 
SoftwareSerial esp8266(2,3); 

void setup() {
  Serial.begin(9600);
  esp8266.begin(115200); // Default baud rate varies
  esp8266.println("AT");
}

void loop() {
  if(esp8266.available()) Serial.write(esp8266.read());
  if(Serial.available()) esp8266.write(Serial.read());
}
