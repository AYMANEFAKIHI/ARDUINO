Project #45: DHT11 Temperature & Humidity

The DHT11 is a very popular, low-cost digital temperature and humidity sensor. It's perfect for basic weather stations, environmental monitoring, and home automation projects. In this project, we will read the values and display them in the Serial Monitor.

1. Concept and Description

The DHT11 is a very popular, low-cost digital temperature and humidity sensor. It's perfect for basic weather stations, environmental monitoring, and home automation projects. In this project, we will read the values and display them in the Serial Monitor.

2. Components Needed

  • Arduino Uno Board
  • USB Cable
  • DHT11 Sensor Module (3-pin or 4-pin)
  • Jumper Wires
  • 10kΩ Resistor (if using the raw 4-pin sensor without a module board)

3. Physical Montage (Wiring Instructions)

If you have the 3-pin module:

  • Signal (S or Data) to Arduino Pin 2
  • VCC (+) to Arduino 5V
  • GND (-) to Arduino GND

If you have the 4-pin sensor:

  • Pin 1 (Left) to 5V
  • Pin 2 to Arduino Pin 2 AND to 5V via a 10kΩ resistor
  • Pin 3 is not connected
  • Pin 4 (Right) to GND

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.


#include "DHT.h"

#define DHTPIN 2     // Digital pin connected to the DHT sensor
#define DHTTYPE DHT11   // DHT 11

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(9600);
  Serial.println(F("DHT11 test!"));

  dht.begin();
}

void loop() {
  // Wait a few seconds between measurements.
  delay(2000);

  // Reading temperature or humidity takes about 250 milliseconds!
  float h = dht.readHumidity();
  // Read temperature as Celsius (the default)
  float t = dht.readTemperature();
  // Read temperature as Fahrenheit (isFahrenheit = true)
  float f = dht.readTemperature(true);

  // Check if any reads failed and exit early (to try again).
  if (isnan(h) || isnan(t) || isnan(f)) {
    Serial.println(F("Failed to read from DHT sensor!"));
    return;
  }

  // Compute heat index in Fahrenheit (the default)
  float hif = dht.computeHeatIndex(f, h);
  // Compute heat index in Celsius (isFahreheit = false)
  float hic = dht.computeHeatIndex(t, h, false);

  Serial.print(F("Humidity: "));
  Serial.print(h);
  Serial.print(F("%  Temperature: "));
  Serial.print(t);
  Serial.print(F("°C "));
  Serial.print(f);
  Serial.print(F("°F  Heat index: "));
  Serial.print(hic);
  Serial.print(F("°C "));
  Serial.print(hif);
  Serial.println(F("°F"));
}
