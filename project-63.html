Project #63: Processing Data Visualizer

Send data from Arduino to Processing (a visual coding language) to create custom graphics.

1. Concept and Description

Processing is a flexible software sketchbook. We can send sensor data to it to create advanced visualizations on the PC screen.

2. Components Needed

  • Arduino Uno
  • Potentiometer

3. Physical Montage (Wiring Instructions)

Potentiometer to A0.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

// Arduino Code
void setup() {
  Serial.begin(9600);
}

void loop() {
  Serial.write(analogRead(A0) / 4); // Send byte (0-255)
  delay(50);
}

/* Processing Code (Run in Processing IDE):
import processing.serial.*;
Serial myPort;
int val;

void setup() {
  size(200, 200);
  myPort = new Serial(this, "COM3", 9600); // Check your COM port
}

void draw() {
  if ( myPort.available() > 0) {
    val = myPort.read();
  }
  background(255);
  fill(0);
  rect(50, 200-val, 100, val);
}
*/
