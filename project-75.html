Project #75: EEPROM Read/Write Settings

Save settings permanently on the Arduino's internal memory.

1. Concept and Description

EEPROM allows you to save small amounts of data (like user settings) that persist even when power is lost.

2. Components Needed

  • Arduino Uno

3. Physical Montage (Wiring Instructions)

None.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.


#include 

int addr = 0;

void setup() {
  Serial.begin(9600);
  
  // Write
  EEPROM.write(addr, 123); 
  
  // Read
  int val = EEPROM.read(addr);
  Serial.print("Value: ");
  Serial.println(val);
}

void loop() {}
