Project #12: Photoresistor (LDR) Night Light

Build an automatic night light that turns on when it gets dark! This project teaches analog light sensing and threshold-based decisions.

1. Concept and Description

In this project, we will explore the following concepts:

  • How photoresistors (LDRs) work
  • Building a voltage divider circuit
  • Setting threshold values for decisions
  • Automatic light-controlled systems

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

/*
 * Project #12: Photoresistor (LDR) Night Light
 * The Arduino Student Lab
 * 
 * Automatically turns on LED when room gets dark
 * Uses analog light sensing with threshold detection
 */

// ============================================
// PIN DEFINITIONS
// ============================================
const int LDR_PIN = A0;      // Analog input for light sensor
const int LED_PIN = 9;       // PWM pin for LED (allows dimming)

// ============================================
// THRESHOLD CONFIGURATION
// ============================================
// Adjust these values based on your lighting conditions!
// Use Serial Monitor to find the right values for your setup
const int DARK_THRESHOLD = 300;   // Below this = turn LED ON
const int LIGHT_THRESHOLD = 500;  // Above this = turn LED OFF
// The gap between thresholds prevents flickering (hysteresis)

// ============================================
// SETUP - Runs once at startup
// ============================================
void setup() {
  pinMode(LED_PIN, OUTPUT);
  // Note: Analog pins don't need pinMode for input
  
  Serial.begin(9600);
  Serial.println("=== AUTOMATIC NIGHT LIGHT ===");
  Serial.println("Cover the LDR to activate the light!");
  Serial.println("");
}

// ============================================
// MAIN LOOP - Runs repeatedly
// ============================================
void loop() {
  // Read light level (0-1023)
  // Low values = dark, High values = bright
  int lightLevel = analogRead(LDR_PIN);
  
  // Print current reading
  Serial.print("Light level: ");
  Serial.print(lightLevel);
  
  // Decide whether to turn light on or off
  if (lightLevel < DARK_THRESHOLD) {
    digitalWrite(LED_PIN, HIGH);
    Serial.println(" - DARK (LED ON)");
  } 
  else if (lightLevel > LIGHT_THRESHOLD) {
    digitalWrite(LED_PIN, LOW);
    Serial.println(" - BRIGHT (LED OFF)");
  }
  else {
    // In the "hysteresis zone" - keep current state
    Serial.println(" - (no change)");
  }
  
  // Small delay for stable readings
  delay(100);
}

 *

 *   Must get brighter than 500 to turn OFF

5. Explanation & Challenges

üîç How It Works

  1. LDR Basics: Resistance decreases as light intensity increases.
  2. Voltage Divider: LDR + fixed resistor create variable voltage at A0.
  3. Analog Reading: Arduino converts 0-5V to 0-1023 digital value.
  4. Hysteresis: Two thresholds prevent LED flickering in borderline light.

üöÄ Challenges

  • Smooth Dimming: Use analogWrite() to make LED brightness proportional to darkness.
  • Auto-Calibration: Read light levels at startup to auto-set thresholds.
  • RGB Night Light: Add RGB LED that slowly cycles colors at night.
  • Motion + Light: Combine with PIR sensor - only turn on when dark AND motion detected.
