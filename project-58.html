Project #58: IR Remote Controlled LED

Control LEDs wirelessly using an IR remote.

1. Concept and Description

Now that we can decode signals, let's use them to control things. Map specific buttons to turn LEDs on or off.

2. Components Needed

  • Arduino Uno
  • IR Receiver
  • Remote
  • LEDs

3. Physical Montage (Wiring Instructions)

Receiver to D11. LED to D13.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.


#include 

const int RECV_PIN = 11;
const int LED_PIN = 13;
IRrecv irrecv(RECV_PIN);
decode_results results;

void setup(){
  irrecv.enableIRIn();
  pinMode(LED_PIN, OUTPUT);
}

void loop(){
  if (irrecv.decode(&results)){
    if (results.value == 0xFFA25D) { // Example code for Power button
      digitalWrite(LED_PIN, !digitalRead(LED_PIN));
    }
    irrecv.resume();
  }
}
