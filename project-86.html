Project #86: MQTT Basics (PubSubClient)

Learn the lightweight messaging protocol used by millions of IoT devices.

1. Concept and Description

MQTT (Message Queuing Telemetry Transport) uses a Publish/Subscribe model. We'll use the PubSubClient library.

2. Components Needed

  • ESP8266
  • MQTT Broker (Mosquitto or Cloud)

3. Physical Montage (Wiring Instructions)

None.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

#include 
#include 

const char* mqtt_server = "broker.mqtt-dashboard.com";
WiFiClient espClient;
PubSubClient client(espClient);

void setup() {
  WiFi.begin("SSID", "PASS");
  client.setServer(mqtt_server, 1883);
}

void loop() {
  if (!client.connected()) {
    // Reconnect logic
  }
  client.loop();
  client.publish("outTopic", "hello world");
  delay(2000);
}
