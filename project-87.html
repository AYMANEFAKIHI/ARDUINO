Project #87: MQTT Home Automation

Control a relay via MQTT messages.

1. Concept and Description

Subscribe to a topic like 'home/livingroom/light' and toggle a relay when a message is received.

2. Components Needed

  • ESP8266
  • Relay

3. Physical Montage (Wiring Instructions)

Relay to D1.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

// Callback function
void callback(char* topic, byte* payload, unsigned int length) {
  if ((char)payload[0] == '1') {
    digitalWrite(RELAY_PIN, HIGH);
  } else {
    digitalWrite(RELAY_PIN, LOW);
  }
}
