Project #44: Relay Module ON/OFF Control

A relay is an electrically operated switch. It allows a low-power signal (like 5V from an Arduino) to control a high-power circuit (like a 12V lamp or even 110V/220V appliances). In this project, we will learn how to toggle a relay safely.

1. Concept and Description

A relay is an electrically operated switch. It allows a low-power signal (like 5V from an Arduino) to control a high-power circuit (like a 12V lamp or even 110V/220V appliances). In this project, we will learn how to toggle a relay safely.

2. Components Needed

  • Arduino Uno Board
  • USB Cable
  • 1-Channel Relay Module (5V)
  • LED (to simulate a high-power load)
  • 220Ω Resistor (for LED)
  • Jumper Wires

3. Physical Montage (Wiring Instructions)

WARNING: Do not attempt to control mains voltage (110V/220V) unless you are a qualified electrician. We will use an LED to simulate a load.

  • Relay VCC to Arduino 5V
  • Relay GND to Arduino GND
  • Relay IN to Arduino Pin 7
  • Relay COM (Common) to Arduino 5V
  • Relay NO (Normally Open) to LED Anode (+)
  • LED Cathode (-) to 220Ω Resistor, then to GND

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

int relayPin = 7;

void setup() {
  pinMode(relayPin, OUTPUT);
}

void loop() {
  // Turn Relay ON
  // Note: Most relay modules are Active LOW
  digitalWrite(relayPin, LOW); 
  delay(2000);
  
  // Turn Relay OFF
  digitalWrite(relayPin, HIGH);
  delay(2000);
}
