Project #77: RFID Access Control System

Build a door lock system that opens only for authorized cards.

1. Concept and Description

Compare the read UID with a stored 'Master' UID to grant or deny access.

2. Components Needed

  • Arduino Uno
  • RC522
  • Servo (Lock)
  • LEDs

3. Physical Montage (Wiring Instructions)

RC522 as before. Servo to D3. LEDs to D4, D5.
