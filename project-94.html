Project #94: Line-Following Robot

Build a robot that autonomously follows a black line on the floor.

1. Concept and Description

Using IR sensors, the robot detects the contrast between the line and the floor.

2. Components Needed

  • Arduino Uno
  • L298N
  • 2 IR Line Sensors
  • Motors

3. Physical Montage (Wiring Instructions)

Left Sensor to D2, Right Sensor to D3.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

// If Left sees Black, Turn Left
// If Right sees Black, Turn Right
// If Both see White, Go Forward
