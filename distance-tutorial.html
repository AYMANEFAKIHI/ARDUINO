HC-SR04 Distance Finder

Build a distance sensor using ultrasonic waves and the pulseIn() function.

1. Concept and Description

Let's give our Arduino "eyes"! The HC-SR04 is an ultrasonic sensor that measures distance by sending out a sound wave and timing how long it takes to bounce back. This project teaches you how to work with time-sensitive sensors using the pulseIn() function and how to perform mathematical calculations within your code to convert raw time data into real-world units like centimeters.

2. Components Needed

  • (1) Arduino Uno
  • (1) HC-SR04 Ultrasonic Sensor
  • (1) Breadboard
  • (4) Jumper Wires

3. Physical Montage (Wiring Instructions)

Follow these steps carefully to wire your circuit.

  1. Place the Sensor: Insert the HC-SR04 into the breadboard facing outward. It has 4 pins: VCC, Trig, Echo, GND.
  2. Power: Connect the **VCC** pin to **5V** on the Arduino. Connect the **GND** pin to **GND** on the Arduino.
  3. Trigger: Connect the **Trig** pin to **Digital Pin 9**. This pin sends the signal to start the sound ping.
  4. Echo: Connect the **Echo** pin to **Digital Pin 10**. This pin listens for the returning sound wave.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE. The comments explain exactly what each line is doing.

// ----------------------------------------------------------------
// HC-SR04 Distance Finder Sketch
// ----------------------------------------------------------------

// Define pins for the ultrasonic sensor
const int trigPin = 9;
const int echoPin = 10;

// Define variables for calculation
long duration; // Variable for the duration of sound wave travel
int distance;  // Variable for the distance measurement

void setup() {
  pinMode(trigPin, OUTPUT); // Sets the trigPin as an Output
  pinMode(echoPin, INPUT);  // Sets the echoPin as an Input
  
  // Start serial communication so we can see the distance on the computer
  Serial.begin(9600); 
}

void loop() {
  // 1. Clear the trigPin condition
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);

  // 2. Sets the trigPin HIGH (ACTIVE) for 10 microseconds
  // This sends out an 8-cycle sonic burst
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);

  // 3. Reads the echoPin, returns the sound wave travel time in microseconds
  duration = pulseIn(echoPin, HIGH);

  // 4. Calculating the distance
  // Speed of sound is roughly 0.034 cm per microsecond.
  // We divide by 2 because the sound goes out AND comes back.
  distance = duration * 0.034 / 2;

  // 5. Print the distance on the Serial Monitor
  Serial.print("Distance: ");
  Serial.print(distance);
  Serial.println(" cm");
  
  // Wait a bit before the next measurement
  delay(100);
}
