Project #49: Rain Sensor Water Detection

Detect rain or water droplets using a rain sensor module.

1. Concept and Description

The rain sensor detects water on its surface. It can be used to trigger an alarm or close a window when it starts raining.

2. Components Needed

  • Arduino Uno
  • Rain Sensor Module
  • Buzzer
  • Jumper Wires

3. Physical Montage (Wiring Instructions)

Sensor VCC to 5V, GND to GND, A0 to A0. Buzzer Positive to D8, Negative to GND.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

const int sensorPin = A0;
const int buzzer = 8;
int sensorValue = 0;

void setup() {
  pinMode(buzzer, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  sensorValue = analogRead(sensorPin);
  Serial.println(sensorValue);
  
  // Lower value means more water
  if(sensorValue < 500) { 
    digitalWrite(buzzer, HIGH);
    delay(100);
    digitalWrite(buzzer, LOW);
    delay(100);
  } else {
    digitalWrite(buzzer, LOW);
  }
  delay(500);
}
