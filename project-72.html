Project #72: PCF8574 I/O Expander

Add more digital input/output pins to your Arduino using I2C.

1. Concept and Description

Running out of pins? The PCF8574 gives you 8 extra I/O pins using only the I2C bus.

2. Components Needed

  • Arduino Uno
  • PCF8574 Module
  • LEDs

3. Physical Montage (Wiring Instructions)

SDA to A4, SCL to A5.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.


#include 
#include "PCF8574.h" // Library required

PCF8574 pcf8574(0x20); // Check address

void setup() {
  pcf8574.begin();
  pcf8574.pinMode(P0, OUTPUT);
}

void loop() {
  pcf8574.digitalWrite(P0, HIGH);
  delay(1000);
  pcf8574.digitalWrite(P0, LOW);
  delay(1000);
}
