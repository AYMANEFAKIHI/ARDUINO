Project #93: Watchdog Timer (WDT)

Automatically reset the Arduino if it freezes or crashes.

1. Concept and Description

The Watchdog Timer is a hardware timer that resets the system if the software gets stuck.

2. Components Needed

  • Arduino Uno

3. Physical Montage (Wiring Instructions)

None.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.


#include 

void setup() {
  wdt_enable(WDTO_2S); // Reset if not patted for 2 seconds
}

void loop() {
  wdt_reset(); // Pat the dog
  // Do tasks
}
