Project #25: Proximity Alert Buzzer

Map distance to pitch! The closer an object gets, the higher the buzzer tone. This creates an intuitive audio feedback system for distance - useful for robotics and accessibility projects.

1. Concept and Description

In this project, we will explore the following concepts:

  • Mapping sensor values to frequency ranges
  • Creating intuitive audio feedback
  • Inverse mapping (closer = higher pitch)
  • Audio-based distance indication

2. Components Needed

  • (1) Arduino Uno
  • (1) HC-SR04 Ultrasonic
  • (1) Piezo Buzzer
  • (1) Breadboard
  • (6+) Jumper Wires

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

/*
 * Project #25: Proximity Alert Buzzer
 * The Arduino Student Lab
 * 
 * Distance mapped to pitch: closer = higher tone
 * Beyond max range = silence
 */

// ============================================
// PIN DEFINITIONS
// ============================================
const int TRIG_PIN = 9;
const int ECHO_PIN = 10;
const int BUZZER_PIN = 8;

// ============================================
// CONFIGURATION
// ============================================
const int MIN_DISTANCE = 5;      // Closest detection (cm)
const int MAX_DISTANCE = 100;    // Farthest detection (cm)
const int MIN_FREQ = 200;        // Low pitch for far objects
const int MAX_FREQ = 2000;       // High pitch for close objects

// ============================================
// SETUP
// ============================================
void setup() {
  pinMode(TRIG_PIN, OUTPUT);
  pinMode(ECHO_PIN, INPUT);
  pinMode(BUZZER_PIN, OUTPUT);
  
  Serial.begin(9600);
  Serial.println("=== PROXIMITY ALERT BUZZER ===");
  Serial.println("Move hand toward sensor to increase pitch");
  Serial.println("");
}

// ============================================
// MAIN LOOP
// ============================================
void loop() {
  // Measure distance
  float distance = measureDistance();
  
  // Check if in range
  if (distance > MAX_DISTANCE || distance < MIN_DISTANCE) {
    // Out of range - silence
    noTone(BUZZER_PIN);
    Serial.print(distance, 1);
    Serial.println(" cm - Out of range (silent)");
  }
  else {
    // Map distance to frequency (inverse relationship)
    // Close (5cm) -> High freq (2000Hz)
    // Far (100cm) -> Low freq (200Hz)
    int frequency = map(distance, MIN_DISTANCE, MAX_DISTANCE, MAX_FREQ, MIN_FREQ);
    
    // Play tone
    tone(BUZZER_PIN, frequency);
    
    // Debug output
    Serial.print(distance, 1);
    Serial.print(" cm -> ");
    Serial.print(frequency);
    Serial.println(" Hz");
  }
  
  delay(50);  // Update rate
}

// ============================================
// HELPER FUNCTIONS
// ============================================

float measureDistance() {
  // Send trigger pulse
  digitalWrite(TRIG_PIN, LOW);
  delayMicroseconds(2);
  digitalWrite(TRIG_PIN, HIGH);
  delayMicroseconds(10);
  digitalWrite(TRIG_PIN, LOW);
  
  // Measure echo
  long duration = pulseIn(ECHO_PIN, HIGH, 30000);
  
  // Calculate distance in cm
  float distance = duration * 0.0343 / 2.0;
  
  // Handle timeout
  if (duration == 0) {
    distance = 999;
  }
  
  return distance;
}

/*
 * MAPPING EXPLANATION
 * 
 * We want: Close = High pitch, Far = Low pitch
 * 
 * map(value, fromLow, fromHigh, toLow, toHigh)
 * 
 * Our mapping:
 *   map(distance, 5, 100, 2000, 200)
 *        ‚îî‚îÄ‚îÄ‚îÄ‚îÄ‚îò   ‚îî‚îÄ‚îÄ‚îÄ‚îÄ‚îò  ‚îî‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îò
 *        input   input    output range
 *        value   range    (REVERSED!)
 * 
 * Example:
 *   distance = 5cm   ‚Üí frequency = 2000 Hz
 *   distance = 52cm  ‚Üí frequency = 1100 Hz
 *   distance = 100cm ‚Üí frequency = 200 Hz
 * 
 * WHY INVERSE MAPPING?
 * - Intuitively, we expect more urgent feedback when close
 * - High pitch = attention, warning
 * - Low pitch = relaxed, safe distance
 * - Same concept used in car parking sensors
 * 
 * FREQUENCY RANGE CHOICES:
 * - 200 Hz: Low, clearly audible, non-alarming
 * - 2000 Hz: High, attention-grabbing
 * - Human hearing: 20 Hz - 20,000 Hz
 * - Best sensitivity: 1,000 - 4,000 Hz
 */

5. Explanation & Challenges

  1. Distance Measurement: HC-SR04 returns distance in centimeters.
  2. Range Check: Ignore readings outside 5-100cm range.
  3. Inverse Mapping: Close distances map to high frequencies.
  4. Continuous Tone: tone() keeps playing until noTone() is called.
