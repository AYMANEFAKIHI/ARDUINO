Project #55: Sound Sensor (KY-038) LED React

Make LEDs flash in sync with music or loud noises using a sound sensor.

1. Concept and Description

The KY-038 sound sensor detects sound intensity. We can use it to create a visualizer or a sound-activated switch.

2. Components Needed

  • Arduino Uno
  • KY-038 Sound Sensor
  • LEDs
  • Resistors

3. Physical Montage (Wiring Instructions)

Sensor AO to A0, G to GND, + to 5V. LED to D13.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

const int sensorPin = A0;
const int ledPin = 13;
int sensorValue = 0;

void setup() {
  pinMode(ledPin, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  sensorValue = analogRead(sensorPin);
  Serial.println(sensorValue);
  
  if (sensorValue > 500) { // Adjust threshold
    digitalWrite(ledPin, HIGH);
    delay(50);
    digitalWrite(ledPin, LOW);
  }
}
