Project #48: Soil Moisture Plant Monitor

Create a system that monitors the moisture level of soil and alerts you when a plant needs watering.

1. Concept and Description

This project uses a soil moisture sensor to detect the water content in the soil. It's the first step towards an automated gardening system.

2. Components Needed

  • Arduino Uno
  • Soil Moisture Sensor Module
  • LEDs (Red, Green)
  • Resistors (220Ω)
  • Jumper Wires

3. Physical Montage (Wiring Instructions)

Sensor VCC to 5V, GND to GND, A0 to A0. Red LED to D13, Green LED to D12.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

const int sensorPin = A0;
const int redLed = 13;
const int greenLed = 12;
int sensorValue = 0;

void setup() {
  pinMode(redLed, OUTPUT);
  pinMode(greenLed, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  sensorValue = analogRead(sensorPin);
  Serial.println(sensorValue);
  
  // Adjust threshold based on your sensor (usually 0-1023, lower is wetter)
  if(sensorValue > 800) { // Dry soil
    digitalWrite(redLed, HIGH);
    digitalWrite(greenLed, LOW);
  } else { // Wet soil
    digitalWrite(redLed, LOW);
    digitalWrite(greenLed, HIGH);
  }
  delay(1000);
}
