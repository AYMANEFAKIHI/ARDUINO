Project #95: Obstacle-Avoiding Robot

A robot that drives around and avoids hitting walls.

1. Concept and Description

Uses an ultrasonic sensor to see obstacles and a servo to look around.

2. Components Needed

  • Arduino Uno
  • Ultrasonic Sensor
  • Servo
  • Motors

3. Physical Montage (Wiring Instructions)

Ultrasonic to D11, D12. Servo to D9.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

// Measure distance
// If distance < 20cm:
//   Stop
//   Look Left, Look Right
//   Turn to open path
