Project #54: Countdown Timer with Buzzer Alarm

Build a countdown timer that sounds an alarm when time is up.

1. Concept and Description

Useful for kitchen timers or games, this project counts down from a set time and triggers a buzzer.

2. Components Needed

  • Arduino Uno
  • 16x2 LCD
  • Buzzer
  • Push Buttons

3. Physical Montage (Wiring Instructions)

LCD as standard. Buzzer to D8. Buttons to D2, D3.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.


#include 

LiquidCrystal lcd(7, 8, 9, 10, 11, 12);
int timeLeft = 60; // Seconds

void setup() {
  lcd.begin(16, 2);
  pinMode(8, OUTPUT);
}

void loop() {
  if (timeLeft > 0) {
    lcd.setCursor(0, 0);
    lcd.print("Time Left: ");
    lcd.print(timeLeft);
    lcd.print(" s  ");
    delay(1000);
    timeLeft--;
  } else {
    lcd.setCursor(0, 0);
    lcd.print("Time's Up!    ");
    digitalWrite(8, HIGH);
    delay(500);
    digitalWrite(8, LOW);
    delay(500);
  }
}
