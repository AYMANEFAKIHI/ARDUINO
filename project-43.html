Project #43: Fan Speed Controller

In this project, we will build a simple temperature-controlled fan system. We will use a potentiometer to simulate a temperature sensor (or you can use a real one if you know how) and adjust the speed of a DC fan accordingly. This is the basic principle behind computer cooling fans.

1. Concept and Description

In this project, we will build a simple temperature-controlled fan system. We will use a potentiometer to simulate a temperature sensor (or you can use a real one if you know how) and adjust the speed of a DC fan accordingly. This is the basic principle behind computer cooling fans.

2. Components Needed

  • Arduino Uno Board
  • USB Cable
  • DC Fan (5V or 12V)
  • NPN Transistor (PN2222 or TIP120)
  • Diode (1N4001)
  • 1kΩ Resistor
  • Potentiometer (to simulate temperature)
  • External Power Supply (if fan requires > 5V or high current)
  • Jumper Wires

3. Physical Montage (Wiring Instructions)

This circuit is very similar to the DC Motor Speed Control project:

  • Transistor Collector to Fan Negative (-)
  • Transistor Emitter to GND
  • Transistor Base to 1kΩ Resistor, then to Arduino Pin 3 (PWM)
  • Fan Positive (+) to Power Supply Positive (+)
  • Power Supply Negative (-) to Arduino GND
  • Diode across Fan terminals (Cathode to Positive)
  • Potentiometer to 5V, GND, and Pin A0

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

int fanPin = 3;      // PWM pin for fan control
int tempPin = A0;    // Potentiometer simulating temp sensor
int tempVal = 0;     // Variable to hold temp reading
int fanSpeed = 0;    // Variable to hold fan speed

void setup() {
  pinMode(fanPin, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  tempVal = analogRead(tempPin); // Read "temperature" (0-1023)
  
  // Let's assume 0-1023 represents 0°C to 100°C roughly
  // We want the fan to start at 30°C (approx val 300) and max out at 70°C (approx val 700)
  
  if (tempVal < 300) {
    fanSpeed = 0; // Too cold, fan off
  } else if (tempVal >= 300 && tempVal <= 700) {
    // Map the range 300-700 to PWM range 50-255
    // We start at 50 because fans often need a minimum kick to start spinning
    fanSpeed = map(tempVal, 300, 700, 50, 255);
  } else {
    fanSpeed = 255; // Too hot, max speed
  }
  
  analogWrite(fanPin, fanSpeed);
  
  Serial.print("Temp Val: ");
  Serial.print(tempVal);
  Serial.print(" | Fan Speed: ");
  Serial.println(fanSpeed);
  
  delay(500);
}
