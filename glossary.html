Quick definitions and syntax references for the most common terms and functions you will encounter in Arduino.

Core Terminology

Sketch
The name given to an Arduino program file (written in C/C++).
Breadboard
A construction base used to build temporary, solderless electronic circuits.
GND (Ground)
Ground (0 Volts). The reference point for all voltage measurements in a circuit.
VCC / 5V
The positive voltage supply pin (usually 5 Volts on Arduino Uno).
PWM (Pulse Width Modulation)
A technique used to simulate analog output (like fading an LED) using a digital pin by quickly turning it on and off. PWM-capable pins are marked with a tilde (~).
Analog vs. Digital
Digital signals are either ON (HIGH/5V) or OFF (LOW/0V). Analog signals can be any value within a range (0-1023 on Arduino).
LED (Light Emitting Diode)
A semiconductor that emits light when current passes through it. Has polarity: longer leg is positive (Anode), shorter leg is negative (Cathode).
Resistor
A component that limits the flow of electrical current. Measured in Ohms (Î©). Used to protect LEDs from burning out.
Pull-Down / Pull-Up Resistor
A resistor used to ensure a known state (LOW or HIGH) for an input pin when no other signal is present, preventing "floating" inputs.
Potentiometer
A three-terminal variable resistor, often used as a control knob (like volume control).

Essential Functions Cheatsheet

Function Purpose Syntax
pinMode() Sets a digital pin to behave as an INPUT or an OUTPUT. pinMode(pin, mode);
digitalWrite() Sets an OUTPUT pin HIGH (5V) or LOW (0V). digitalWrite(pin, value);
digitalRead() Reads the state of an INPUT pin (returns HIGH or LOW). digitalRead(pin);
analogRead() Reads a voltage from an Analog pin (A0-A5) and returns a value between 0 and 1023. analogRead(pin);
analogWrite() Outputs a PWM signal (0-255) to simulate analog voltage on a PWM pin (~). analogWrite(pin, value);
delay() Pauses the program for a specified number of milliseconds (ms). delay(ms);
delayMicroseconds() Pauses the program for a specified number of microseconds (Î¼s). delayMicroseconds(us);
map() Scales a number from one range to another (e.g., 0-1023 to 0-255). map(value, fromLow, fromHigh, toLow, toHigh);
pulseIn() Measures the duration of a pulse (HIGH or LOW) on a pin in microseconds. pulseIn(pin, value);
Serial.begin() Initializes serial communication at a specified baud rate. Serial.begin(9600);
Serial.print() Sends data to the Serial Monitor without a newline. Serial.print(data);
Serial.println() Sends data to the Serial Monitor followed by a newline. Serial.println(data);

Program Structure

Every Arduino sketch has two required functions:

void setup()
Runs once when the Arduino is powered on or reset. Used to initialize pins, start serial communication, etc.
void loop()
Runs repeatedly forever after setup() completes. This is where your main program logic goes.
