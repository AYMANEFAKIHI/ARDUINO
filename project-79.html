Project #79: ESP8266 Web Server LED Control

Control an LED from a web page hosted on the ESP8266.

1. Concept and Description

Turn your ESP8266 into a mini web server that serves a page with buttons.

2. Components Needed

  • ESP8266 (NodeMCU recommended for ease)
  • LED

3. Physical Montage (Wiring Instructions)

LED to D1 (GPIO 5).

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.
