Project #61: Serial LED Control (PC Commands)

Control an LED by typing commands on your computer keyboard.

1. Concept and Description

Learn how to send data from your computer to the Arduino via the Serial Monitor to control hardware.

2. Components Needed

  • Arduino Uno
  • LED
  • Resistor (220Ω)

3. Physical Montage (Wiring Instructions)

LED Anode to D13, Cathode to GND (via resistor).

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

void setup() {
  pinMode(13, OUTPUT);
  Serial.begin(9600);
  Serial.println("Type '1' to turn ON, '0' to turn OFF");
}

void loop() {
  if (Serial.available() > 0) {
    char command = Serial.read();
    if (command == '1') {
      digitalWrite(13, HIGH);
      Serial.println("LED ON");
    } else if (command == '0') {
      digitalWrite(13, LOW);
      Serial.println("LED OFF");
    }
  }
}
