Fading LED with a Potentiometer

Explore analog inputs, PWM output, and mapping values with a potentiometer.

1. Concept and Description

Digital signals are just On or Off, but the real world is analog (variable). In this project, we will use a Potentiometer (a variable resistor, like a volume knob) to control the brightness of an LED. This introduces **Analog Input** (reading a range of values), **PWM (Pulse Width Modulation)** to simulate analog output, and the very useful map() function to translate numbers from one range to another.

2. Components Needed

  • (1) Arduino Uno
  • (1) Potentiometer (10k $\Omega$ is standard)
  • (1) LED
  • (1) 220 $\Omega$ Resistor
  • (1) Breadboard
  • Jumper Wires

3. Physical Montage (Wiring Instructions)

Follow these steps carefully to wire your circuit.

  1. LED Setup: Connect the LED and 220 $\Omega$ resistor as usual, but this time connect the Anode (long leg) to **Digital Pin 9**.
    Note: Pin 9 has a "~" symbol next to it, meaning it supports PWM (fading).
  2. Place the Potentiometer: Insert the potentiometer into the breadboard. It has three pins.
  3. Power the Potentiometer: Connect the **Left** outer pin to **5V**. Connect the **Right** outer pin to **GND**.
  4. Connect the Wiper: Connect the **Middle** pin (the wiper) to **Analog Pin A0** on the Arduino.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE. The comments explain exactly what each line is doing.

// ----------------------------------------------------------------
// Fading LED with Potentiometer Sketch
// ----------------------------------------------------------------

const int potPin = A0;  // Analog input pin that the potentiometer is attached to
const int ledPin = 9;   // Analog output (PWM) pin that the LED is attached to

int sensorValue = 0;    // Value read from the pot
int outputValue = 0;    // Value output to the PWM (analog out)

void setup() {
  // No pinMode needed for Analog Input (A0), it is automatic.
  // We set the LED pin as output.
  pinMode(ledPin, OUTPUT);
  
  // Initialize serial communications at 9600 bps (optional, for debugging):
  Serial.begin(9600);
}

void loop() {
  // Read the analog in value:
  // This returns a value between 0 (0V) and 1023 (5V)
  sensorValue = analogRead(potPin);

  // Map it to the range of the analog out:
  // The analogWrite function only accepts values between 0 and 255.
  // We use map(value, fromLow, fromHigh, toLow, toHigh)
  outputValue = map(sensorValue, 0, 1023, 0, 255);

  // Change the analog out value:
  analogWrite(ledPin, outputValue);

  // Print the results to the Serial Monitor (Tools > Serial Monitor)
  Serial.print("sensor = ");
  Serial.print(sensorValue);
  Serial.print("\t output = ");
  Serial.println(outputValue);

  // Wait 2 milliseconds before the next loop for the ADC to settle
  delay(2);
}
