Project #50: MQ-2 Gas/Smoke Detector

Build a safety device that detects smoke or flammable gases like LPG, butane, and propane.

1. Concept and Description

The MQ-2 sensor is sensitive to LPG, i-butane, propane, methane, alcohol, hydrogen, and smoke. It's widely used in gas leakage detecting equipment.

2. Components Needed

  • Arduino Uno
  • MQ-2 Gas Sensor
  • Buzzer
  • Red LED
  • Jumper Wires

3. Physical Montage (Wiring Instructions)

Sensor VCC to 5V, GND to GND, A0 to A0. Buzzer to D8, LED to D9.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

const int sensorPin = A0;
const int buzzer = 8;
const int led = 9;
int sensorValue = 0;

void setup() {
  pinMode(buzzer, OUTPUT);
  pinMode(led, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  sensorValue = analogRead(sensorPin);
  Serial.println(sensorValue);
  
  if(sensorValue > 300) { // Threshold depends on environment
    digitalWrite(buzzer, HIGH);
    digitalWrite(led, HIGH);
  } else {
    digitalWrite(buzzer, LOW);
    digitalWrite(led, LOW);
  }
  delay(500);
}
