Project #8: Tilt Switch Orientation Sensor

Detect orientation changes with a simple tilt switch! This ball-bearing sensor closes a circuit when tilted, making it perfect for detecting movement or position changes.

1. Concept and Description

In this project, we will explore the following concepts:

  • How tilt switches work (ball-bearing mechanism)
  • Using different types of digital input switches
  • Building orientation-sensitive projects
  • Debouncing mechanical sensors

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

/*
 * Project #8: Tilt Switch Orientation Sensor
 * The Arduino Student Lab
 * 
 * Detects when the sensor is tilted past its trigger angle
 * Uses debouncing for stable readings
 */

// ============================================
// PIN DEFINITIONS
// ============================================
const int TILT_PIN = 2;
const int LED_PIN = 13;

// ============================================
// DEBOUNCE CONFIGURATION
// ============================================
const unsigned long DEBOUNCE_DELAY = 100;  // Tilt switches need more debounce time

// ============================================
// STATE VARIABLES
// ============================================
bool tiltState = false;
bool lastTiltState = false;
unsigned long lastDebounceTime = 0;

// ============================================
// SETUP - Runs once at startup
// ============================================
void setup() {
  pinMode(TILT_PIN, INPUT);
  pinMode(LED_PIN, OUTPUT);
  
  Serial.begin(9600);
  Serial.println("Tilt Sensor Ready!");
  Serial.println("Tilt the sensor to trigger the LED.");
}

// ============================================
// MAIN LOOP - Runs repeatedly
// ============================================
void loop() {
  // Read current tilt sensor state
  bool reading = digitalRead(TILT_PIN);
  
  // Check if state changed
  if (reading != lastTiltState) {
    lastDebounceTime = millis();
  }
  
  // If stable for DEBOUNCE_DELAY, accept new state
  if ((millis() - lastDebounceTime) > DEBOUNCE_DELAY) {
    if (reading != tiltState) {
      tiltState = reading;
      
      // Update LED to match tilt state
      digitalWrite(LED_PIN, tiltState);
      
      // Debug output
      if (tiltState) {
        Serial.println("TILTED! (circuit closed)");
      } else {
        Serial.println("LEVEL (circuit open)");
      }
    }
  }
  
  lastTiltState = reading;
}

/*
 * HOW TILT SWITCHES WORK
 * 
 * Inside the switch is a small metal ball bearing:
 * 
 *   Upright:          Tilted:
 *   +------+          +------+
 *   |      |          |    o |  <- ball rolls
 *   |  o   |          |   /  |
 *   +--||--+          +--||--+
 *      ||                ||
 *   contacts           contacts touched!
 * 
 * When tilted past the threshold angle (~15-45¬∞),
 * the ball touches both contacts, closing the circuit.
 * 
 * APPLICATIONS:
 * - Detect if a device has been knocked over
 * - Motion-activated lights
 * - Gaming controllers (tilt steering)
 * - Anti-theft alarms
 */

5. Explanation & Challenges

üîç How It Works

  1. Ball Bearing: A metal ball inside the switch rolls with gravity.
  2. Contact Points: When tilted, the ball bridges two contact points.
  3. Digital Signal: Circuit closed = HIGH, circuit open = LOW.
  4. Debouncing: The ball can bounce, so we wait 100ms for stable readings.

üöÄ Challenges

  • Four-Way Detection: Use 4 tilt switches to detect North/South/East/West orientation.
  • Motion Alarm: Sound a buzzer if the sensor detects any tilt.
  • Game Controller: Use tilt to control LED brightness (tilt more = brighter).
  • Level Indicator: Use 2 tilt switches to create a simple level/bubble indicator.
