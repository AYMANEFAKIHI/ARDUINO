Project #80: ESP8266 Access Point (AP) Mode

Make the ESP8266 create its own Wi-Fi network that you can connect to.

1. Concept and Description

Useful when no router is available. The ESP becomes the router.

2. Components Needed

  • ESP8266

3. Physical Montage (Wiring Instructions)

None.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

#include 

void setup() {
  WiFi.softAP("MyESPNetwork", "12345678");
}

void loop() {}
