Project #9: Magnetic Reed Switch Door Alarm

Build a door/window security sensor! Reed switches activate when a magnet is nearby - the same technology used in real home security systems.

1. Concept and Description

In this project, we will explore the following concepts:

  • How magnetic reed switches work
  • Building proximity detection circuits
  • Creating security/alarm systems
  • Using normally-open (NO) vs normally-closed (NC) switches

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

/*
 * Project #9: Magnetic Reed Switch Door Alarm
 * The Arduino Student Lab
 * 
 * Detects when a magnet is removed (door opened)
 * Triggers LED and optional buzzer alarm
 */

// ============================================
// PIN DEFINITIONS
// ============================================
const int REED_PIN = 2;      // Reed switch input
const int LED_PIN = 13;      // Alarm LED
const int BUZZER_PIN = 8;    // Optional buzzer

// ============================================
// ALARM CONFIGURATION
// ============================================
const int ALARM_FREQUENCY = 2000;  // Buzzer frequency in Hz
const int ALARM_DURATION = 100;    // Beep duration in ms

// ============================================
// STATE VARIABLES
// ============================================
bool doorClosed = true;      // Magnet present = door closed
bool alarmArmed = true;      // Is alarm system active?

// ============================================
// SETUP - Runs once at startup
// ============================================
void setup() {
  pinMode(REED_PIN, INPUT);
  pinMode(LED_PIN, OUTPUT);
  pinMode(BUZZER_PIN, OUTPUT);
  
  Serial.begin(9600);
  Serial.println("=== DOOR ALARM SYSTEM ===");
  Serial.println("Place magnet near reed switch.");
  Serial.println("Remove magnet to trigger alarm.");
  Serial.println("");
  
  // Check initial state
  doorClosed = digitalRead(REED_PIN);
  if (doorClosed) {
    Serial.println("STATUS: Door CLOSED (magnet detected)");
  } else {
    Serial.println("STATUS: Door OPEN (no magnet)");
  }
}

// ============================================
// MAIN LOOP - Runs repeatedly
// ============================================
void loop() {
  // Read reed switch state
  // HIGH = magnet present (door closed)
  // LOW = magnet absent (door open)
  bool currentState = digitalRead(REED_PIN);
  
  // Check for state change
  if (currentState != doorClosed) {
    doorClosed = currentState;
    
    if (doorClosed) {
      // Door just closed
      Serial.println(">> Door CLOSED - All clear");
      digitalWrite(LED_PIN, LOW);
      noTone(BUZZER_PIN);
    } else {
      // Door just opened!
      Serial.println("!! ALERT: Door OPENED !!");
      
      if (alarmArmed) {
        triggerAlarm();
      }
    }
  }
  
  // While door is open and alarm is armed, keep alerting
  if (!doorClosed && alarmArmed) {
    // Blink LED
    digitalWrite(LED_PIN, HIGH);
    tone(BUZZER_PIN, ALARM_FREQUENCY, ALARM_DURATION);
    delay(200);
    digitalWrite(LED_PIN, LOW);
    delay(200);
  }
}

// ============================================
// HELPER FUNCTIONS
// ============================================

void triggerAlarm() {
  Serial.println("** ALARM TRIGGERED **");
  
  // Initial alarm burst
  for (int i = 0; i < 5; i++) {
    digitalWrite(LED_PIN, HIGH);
    tone(BUZZER_PIN, ALARM_FREQUENCY);
    delay(100);
    digitalWrite(LED_PIN, LOW);
    noTone(BUZZER_PIN);
    delay(100);
  }
}

/*
 * HOW REED SWITCHES WORK
 * 
 * Inside the glass tube are two flexible metal "reeds":
 * 
 *   Without Magnet:       With Magnet:
 *   +-------------+       +-------------+
 *   |   /    \    |       |   /----\    |
 *   |  |      |   |  -->  |  |      |   |
 *   |   \    /    |       |   \----/    |
 *   +-------------+       +-------------+
 *      Gap (open)           Touching (closed)
 * 
 * The magnetic field pulls the reeds together!
 * 
 * REAL-WORLD USES:
 * - Door/window security sensors
 * - Laptop lid detection
 * - Bicycle speedometers
 * - Position sensing in machinery
 */

5. Explanation & Challenges

üîç How It Works

  1. Reed Switch: Two metal contacts in a sealed glass tube respond to magnetic fields.
  2. Magnet Proximity: When magnet is close, contacts touch (circuit closed).
  3. Door Simulation: Magnet on door, switch on frame - separating them triggers alarm.
  4. Continuous Alert: While door is open, LED blinks and buzzer sounds.

üöÄ Challenges

  • Arm/Disarm: Add a button or keypad to arm/disarm the alarm.
  • Entry Delay: Give 30 seconds to enter a code before alarm sounds.
  • Multiple Zones: Add reed switches for multiple doors/windows.
  • SMS Alert: Later, add WiFi to send text alerts when triggered.
