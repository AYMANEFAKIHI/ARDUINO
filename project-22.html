Project #22: PIR Motion-Activated Light

Build an automatic security light that turns on when motion is detected! PIR (Passive Infrared) sensors detect body heat - the same technology used in home security systems.

1. Concept and Description

In this project, we will explore the following concepts:

  • How PIR motion sensors work
  • Implementing timeout logic
  • Building automatic lighting systems
  • Adjusting sensor sensitivity and delay

2. Components Needed

  • (1) HC-SR501 PIR Sensor
  • (1) LED

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

/*
 * Project #22: PIR Motion-Activated Light
 * The Arduino Student Lab
 * 
 * Automatically turns on light when motion detected
 * Stays on for configured duration, then turns off
 */

// ============================================
// PIN DEFINITIONS
// ============================================
const int PIR_PIN = 2;       // PIR sensor output
const int LED_PIN = 13;      // Light to control

// ============================================
// TIMING CONFIGURATION
// ============================================
const unsigned long LIGHT_DURATION = 10000;  // Stay on for 10 seconds
const unsigned long WARMUP_TIME = 30000;     // PIR warmup period (30s)

// ============================================
// STATE VARIABLES
// ============================================
bool lightOn = false;
unsigned long lightTurnedOnTime = 0;
bool sensorReady = false;
int motionCount = 0;

// ============================================
// SETUP - Runs once at startup
// ============================================
void setup() {
  pinMode(PIR_PIN, INPUT);
  pinMode(LED_PIN, OUTPUT);
  
  digitalWrite(LED_PIN, LOW);
  
  Serial.begin(9600);
  Serial.println("=== PIR MOTION-ACTIVATED LIGHT ===");
  Serial.println("");
  Serial.println("PIR sensor warming up...");
  Serial.println("Please wait 30 seconds for calibration.");
  Serial.println("");
  
  // Blink LED during warmup
  for (int i = 0; i < 30; i++) {
    digitalWrite(LED_PIN, HIGH);
    delay(100);
    digitalWrite(LED_PIN, LOW);
    delay(900);
    Serial.print(30 - i);
    Serial.println(" seconds remaining...");
  }
  
  sensorReady = true;
  Serial.println("");
  Serial.println("Sensor ready! Detecting motion...");
  Serial.println("");
}

// ============================================
// MAIN LOOP - Runs repeatedly
// ============================================
void loop() {
  if (!sensorReady) return;
  
  unsigned long currentTime = millis();
  
  // Read PIR sensor
  bool motionDetected = digitalRead(PIR_PIN);
  
  if (motionDetected) {
    if (!lightOn) {
      // First motion detection - turn on light
      motionCount++;
      lightOn = true;
      lightTurnedOnTime = currentTime;
      digitalWrite(LED_PIN, HIGH);
      
      Serial.print("[");
      Serial.print(currentTime / 1000);
      Serial.print("s] Motion #");
      Serial.print(motionCount);
      Serial.println(" detected! Light ON");
    } else {
      // Motion while light is on - reset timer
      lightTurnedOnTime = currentTime;
      Serial.println("  Motion continues - timer reset");
    }
  }
  
  // Check if light should turn off
  if (lightOn && (currentTime - lightTurnedOnTime >= LIGHT_DURATION)) {
    lightOn = false;
    digitalWrite(LED_PIN, LOW);
    
    Serial.print("[");
    Serial.print(currentTime / 1000);
    Serial.println("s] No motion - Light OFF");
    Serial.println("");
  }
  
  delay(100);  // Small delay to reduce CPU usage
}

/*
 * HOW PIR SENSORS WORK
 * 
 * PIR = Passive Infrared
 * "Passive" because it doesn't emit anything - only receives
 * 
 * Inside the sensor:
 * +---------------------------+
 * |  [Fresnel Lens]           |  <- Focuses IR from wide area
 * |      |                    |
 * |  [Pyroelectric Sensor]    |  <- Detects IR changes
 * |      |                    |
 * |  [Signal Processing]      |  <- Amplifies & compares
 * +---------------------------+
 * 
 * Detection principle:
 * - All warm objects emit infrared radiation
 * - Humans emit IR around 10 micrometers wavelength
 * - Sensor has two elements that detect IR
 * - When a person moves, one element sees more IR
 * - The difference triggers detection
 * 
 * HC-SR501 ADJUSTMENTS:
 * - Left pot: Sensitivity (clockwise = more sensitive)
 * - Right pot: Time delay (clockwise = longer)
 * - Jumper: H = retriggerable, L = single trigger
 * 
 * WARMUP PERIOD:
 * PIR sensors need 30-60 seconds to calibrate to
 * ambient IR levels. May give false triggers during warmup.
 */

5. Explanation & Challenges

üîç How It Works

  1. PIR Detection: Sensor outputs HIGH when motion (IR change) detected.
  2. Warmup: 30-second calibration period prevents false triggers.
  3. Timer Reset: Continued motion resets the off-timer.
  4. Auto-Off: Light turns off after no motion for set duration.

üöÄ Challenges

  • Daylight Override: Add LDR to only activate at night.
  • Adjustable Duration: Use potentiometer to set light-on time.
  • Multiple Zones: Add more PIR sensors for different areas.
  • Real Light: Use a relay module to control an AC lamp.
