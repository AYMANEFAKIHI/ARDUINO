Project #46: DHT22 Weather Station (LCD)

Build a weather station that displays temperature and humidity readings from a DHT22 sensor on an LCD screen.

1. Concept and Description

In this project, we will combine the DHT22 temperature and humidity sensor with an LCD display to create a standalone weather station. The DHT22 is more accurate than the DHT11 used in earlier projects.

2. Components Needed

  • Arduino Uno
  • DHT22 Sensor
  • 16x2 LCD Display
  • Potentiometer (10k)
  • Jumper Wires
  • Breadboard

3. Physical Montage (Wiring Instructions)

DHT22: Pin 1 to 5V, Pin 2 to D2, Pin 4 to GND. (Add 10k pull-up between Pin 1 and 2 if module doesn't have one).
LCD: RS to D7, E to D8, D4 to D9, D5 to D10, D6 to D11, D7 to D12. VSS to GND, VDD to 5V, V0 to Potentiometer.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.


#include 
#include "DHT.h"

#define DHTPIN 2     
#define DHTTYPE DHT22   

DHT dht(DHTPIN, DHTTYPE);
LiquidCrystal lcd(7, 8, 9, 10, 11, 12);

void setup() {
  lcd.begin(16, 2);
  dht.begin();
}

void loop() {
  delay(2000);

  float h = dht.readHumidity();
  float t = dht.readTemperature();

  if (isnan(h) || isnan(t)) {
    lcd.setCursor(0, 0);
    lcd.print("Failed to read");
    return;
  }

  lcd.setCursor(0, 0);
  lcd.print("Temp: ");
  lcd.print(t);
  lcd.print(" C");

  lcd.setCursor(0, 1);
  lcd.print("Hum:  ");
  lcd.print(h);
  lcd.print(" %");
}
