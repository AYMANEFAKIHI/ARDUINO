Project #15: Breathing LED (Smooth Fade)

Create the iconic "breathing" LED effect seen on sleeping MacBooks! This project uses sine wave math for an organic, smooth pulsing animation.

1. Concept and Description

In this project, we will explore the following concepts:

  • Using PWM for smooth LED dimming
  • Applying math functions (sin()) to create natural animations
  • Understanding radians vs degrees
  • Creating non-linear fade effects

/*
 * Project #15: Breathing LED (Smooth Fade)
 * The Arduino Student Lab
 * 
 * Creates a smooth, organic "breathing" animation
 * using sine wave mathematics
 */

// ============================================
// PIN DEFINITIONS
// ============================================
const int LED_PIN = 9;  // Must be a PWM pin (~)

// ============================================
// ANIMATION CONFIGURATION
// ============================================
const float BREATH_SPEED = 0.02;   // How fast to breathe (smaller = slower)
const int DELAY_TIME = 10;         // Milliseconds between updates

// ============================================
// STATE VARIABLES
// ============================================
float angle = 0;  // Current angle for sine wave (in radians)

// ============================================
// SETUP - Runs once at startup
// ============================================
void setup() {
  pinMode(LED_PIN, OUTPUT);
  
  Serial.begin(9600);
  Serial.println("=== BREATHING LED ===");
  Serial.println("Watch the LED pulse smoothly!");
}

// ============================================
// MAIN LOOP - Runs repeatedly
// ============================================
void loop() {
  // Calculate brightness using sine wave
  // sin() returns -1 to +1, we need 0 to 255
  
  // Step 1: Get sine value (-1 to +1)
  float sineValue = sin(angle);
  
  // Step 2: Shift to 0 to +2 range
  float shifted = sineValue + 1;
  
  // Step 3: Scale to 0-255 (PWM range)
  int brightness = shifted * 127.5;
  
  // Apply brightness to LED
  analogWrite(LED_PIN, brightness);
  
  // Increment angle for next frame
  angle += BREATH_SPEED;
  
  // Keep angle from getting too large (optional but good practice)
  if (angle >= 2 * PI) {
    angle = 0;
  }
  
  // Debug output (every 50 loops to avoid flooding)
  static int loopCount = 0;
  if (loopCount++ % 50 == 0) {
    Serial.print("Angle: ");
    Serial.print(angle, 2);
    Serial.print(" | Sine: ");
    Serial.print(sineValue, 2);
    Serial.print(" | Brightness: ");
    Serial.println(brightness);
  }
  
  delay(DELAY_TIME);
}

5. Explanation & Challenges

üîç How It Works

  1. Sine Wave: sin() produces smooth oscillation between -1 and +1.
  2. Normalization: Add 1 to shift range to 0-2, then multiply by 127.5 for 0-255.
  3. PWM Output: analogWrite() sets LED brightness (0=off, 255=full).
  4. Animation Loop: Increment angle continuously, sine value changes smoothly.

üöÄ Challenges

  • Adjustable Speed: Use a potentiometer to control breathing rate.
  • RGB Breathing: Three LEDs breathing at different rates for color mixing.
  • Heartbeat: Modify the wave for a double-pulse heartbeat pattern.
  • Multiple LEDs: Phase-offset multiple LEDs for a "chasing pulse" effect.
