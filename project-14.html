Project #14: 5-LED Volume/Level Meter

Create a visual level indicator using 5 LEDs! Turn a potentiometer and watch LEDs light up sequentially - like a VU meter or signal strength indicator.

1. Concept and Description

In this project, we will explore the following concepts:

  • Mapping analog values to discrete levels
  • Controlling multiple outputs based on input ranges
  • Building bar graph displays
  • Using arrays for efficient LED control

  • (1) Arduino Uno
  • (5) LEDs
  • (1) Potentiometer

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

/*
 * Project #14: 5-LED Volume/Level Meter
 * The Arduino Student Lab
 * 
 * Displays analog input level as a bar graph
 * using 5 LEDs (like a VU meter)
 */

// ============================================
// PIN DEFINITIONS
// ============================================
const int POT_PIN = A0;        // Potentiometer input
const int NUM_LEDS = 5;        // Number of LEDs in bar graph

// LED pins in order from lowest to highest
const int LED_PINS[NUM_LEDS] = {2, 3, 4, 5, 6};

// ============================================
// THRESHOLD VALUES
// ============================================
// Divide 0-1023 range into 5 equal segments
// Each segment is ~205 units wide
const int THRESHOLDS[NUM_LEDS] = {205, 410, 615, 820, 1000};

// ============================================
// SETUP - Runs once at startup
// ============================================
void setup() {
  // Set all LED pins as outputs
  for (int i = 0; i < NUM_LEDS; i++) {
    pinMode(LED_PINS[i], OUTPUT);
  }
  
  Serial.begin(9600);
  Serial.println("=== 5-LED LEVEL METER ===");
  Serial.println("Turn the potentiometer to see LEDs respond!");
}

// ============================================
// MAIN LOOP - Runs repeatedly
// ============================================
void loop() {
  // Read potentiometer value (0-1023)
  int sensorValue = analogRead(POT_PIN);
  
  // Determine how many LEDs should be lit
  int ledsToLight = 0;
  for (int i = 0; i < NUM_LEDS; i++) {
    if (sensorValue >= THRESHOLDS[i]) {
      ledsToLight = i + 1;
    }
  }
  
  // Update LEDs
  updateLEDs(ledsToLight);
  
  // Debug output
  Serial.print("Value: ");
  Serial.print(sensorValue);
  Serial.print(" | Level: ");
  Serial.print(ledsToLight);
  Serial.print("/5 ");
  printBar(ledsToLight);
  Serial.println();
  
  delay(50);  // Small delay for stable readings
}

// ============================================
// HELPER FUNCTIONS
// ============================================

/*
 * updateLEDs(count)
 * Lights up 'count' LEDs from the bottom up
 * Turns off LEDs above that count
 */
void updateLEDs(int count) {
  for (int i = 0; i < NUM_LEDS; i++) {
    if (i < count) {
      digitalWrite(LED_PINS[i], HIGH);  // Light this LED
    } else {
      digitalWrite(LED_PINS[i], LOW);   // Turn off this LED
    }
  }
}

/*
 * printBar(level)
 * Prints a text-based bar graph to Serial Monitor
 */
void printBar(int level) {
  Serial.print("[");
  for (int i = 0; i < NUM_LEDS; i++) {
    if (i < level) {
      Serial.print("‚ñà");  // Filled block
    } else {
      Serial.print("‚ñë");  // Empty block
    }
  }
  Serial.print("]");
}

/*
 * UNDERSTANDING THE THRESHOLDS:
 * 
 * Analog range: 0 to 1023
 * 
 * Value:     0    205   410   615   820   1023
 *            |-----|-----|-----|-----|-----|
 * LEDs lit:    0     1     2     3     4     5
 * 
 * LED 1 lights when value >= 1 (almost immediately)
 * LED 2 lights when value >= 205 (20%)
 * LED 3 lights when value >= 410 (40%)
 * LED 4 lights when value >= 615 (60%)
 * LED 5 lights when value >= 820 (80%)
 * 
 * You can adjust thresholds for non-linear response!
 */

5. Explanation & Challenges

üîç How It Works

  1. Analog Input: Potentiometer outputs 0-1023 based on position.
  2. Thresholds: We divide the range into 5 segments (0-20%, 20-40%, etc.).
  3. LED Count: Count how many thresholds the value exceeds.
  4. Bar Graph: Light LEDs from bottom up based on the count.

üöÄ Challenges

  • Sound Meter: Replace potentiometer with a sound sensor for a VU meter.
  • Peak Hold: Keep the highest LED lit for 1 second before dropping.
  • 10-LED Bar: Expand to 10 LEDs for higher resolution.
  • RGB Colors: Use NeoPixels for a smooth color-changing bar graph.
