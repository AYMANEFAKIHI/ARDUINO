Project #16: Simple Melody with Piezo Buzzer

Make your Arduino play music! Using the tone() function and musical note frequencies, create melodies that would make Beethoven proud (or at least mildly amused).

1. Concept and Description

In this project, we will explore the following concepts:

  • How piezo buzzers produce sound
  • Using the tone() and noTone() functions
  • Musical note frequencies
  • Storing melodies in arrays

2. Components Needed

  • (1) Arduino Uno
  • (1) Piezo Buzzer
  • (1) Breadboard
  • (2) Jumper Wires

3. Physical Montage (Wiring Instructions)

Pin Connections:

Arduino PinComponent
Pin 8Buzzer positive (+)
GNDBuzzer negative (-)

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

/*
 * Project #16: Simple Melody with Piezo Buzzer
 * The Arduino Student Lab
 * 
 * Plays "Twinkle Twinkle Little Star" using tone()
 */

// ============================================
// PIN DEFINITIONS
// ============================================
const int BUZZER_PIN = 8;

// ============================================
// MUSICAL NOTE FREQUENCIES (Hz)
// ============================================
#define NOTE_C4  262   // Middle C
#define NOTE_D4  294
#define NOTE_E4  330
#define NOTE_F4  349
#define NOTE_G4  392
#define NOTE_A4  440
#define NOTE_B4  494
#define NOTE_C5  523   // C one octave up
#define NOTE_REST 0    // Silence

// ============================================
// MELODY DEFINITION
// ============================================
// Twinkle Twinkle Little Star
int melody[] = {
  NOTE_C4, NOTE_C4, NOTE_G4, NOTE_G4, NOTE_A4, NOTE_A4, NOTE_G4,  // Twin-kle twin-kle lit-tle star
  NOTE_F4, NOTE_F4, NOTE_E4, NOTE_E4, NOTE_D4, NOTE_D4, NOTE_C4,  // How I won-der what you are
  NOTE_G4, NOTE_G4, NOTE_F4, NOTE_F4, NOTE_E4, NOTE_E4, NOTE_D4,  // Up a-bove the world so high
  NOTE_G4, NOTE_G4, NOTE_F4, NOTE_F4, NOTE_E4, NOTE_E4, NOTE_D4,  // Like a dia-mond in the sky
  NOTE_C4, NOTE_C4, NOTE_G4, NOTE_G4, NOTE_A4, NOTE_A4, NOTE_G4,  // Twin-kle twin-kle lit-tle star
  NOTE_F4, NOTE_F4, NOTE_E4, NOTE_E4, NOTE_D4, NOTE_D4, NOTE_C4   // How I won-der what you are
};

// Note durations: 4 = quarter note, 8 = eighth note, etc.
int noteDurations[] = {
  4, 4, 4, 4, 4, 4, 2,
  4, 4, 4, 4, 4, 4, 2,
  4, 4, 4, 4, 4, 4, 2,
  4, 4, 4, 4, 4, 4, 2,
  4, 4, 4, 4, 4, 4, 2,
  4, 4, 4, 4, 4, 4, 2
};

// Calculate number of notes
const int NUM_NOTES = sizeof(melody) / sizeof(melody[0]);

// Tempo control (higher = faster)
const int TEMPO = 120;  // BPM (beats per minute)

// ============================================
// SETUP - Runs once at startup
// ============================================
void setup() {
  pinMode(BUZZER_PIN, OUTPUT);
  
  Serial.begin(9600);
  Serial.println("=== MELODY PLAYER ===");
  Serial.println("Playing Twinkle Twinkle Little Star...");
  Serial.println("");
}

// ============================================
// MAIN LOOP - Runs repeatedly
// ============================================
void loop() {
  // Play each note in the melody
  for (int i = 0; i < NUM_NOTES; i++) {
    // Calculate note duration
    // Quarter note (4) at 120 BPM = 500ms
    int noteDuration = (60000 / TEMPO) / (noteDurations[i] / 4.0);
    
    // Play the note
    if (melody[i] != NOTE_REST) {
      tone(BUZZER_PIN, melody[i], noteDuration);
      Serial.print("Note ");
      Serial.print(i + 1);
      Serial.print(": ");
      Serial.print(melody[i]);
      Serial.println(" Hz");
    }
    
    // Pause between notes (adds slight separation)
    int pauseBetweenNotes = noteDuration * 1.3;
    delay(pauseBetweenNotes);
    
    // Stop the tone
    noTone(BUZZER_PIN);
  }
  
  // Pause before repeating
  Serial.println("\n--- Melody complete! Restarting... ---\n");
  delay(2000);
}

/*
 * MUSICAL FREQUENCY REFERENCE
 * 
 * Note  | Freq (Hz)
 * ------|----------
 * C4    | 262
 * C#4   | 277
 * D4    | 294
 * D#4   | 311
 * E4    | 330
 * F4    | 349
 * F#4   | 370
 * G4    | 392
 * G#4   | 415
 * A4    | 440  (Standard tuning reference)
 * A#4   | 466
 * B4    | 494
 * C5    | 523
 * 
 * Each octave doubles the frequency:
 * A3 = 220 Hz, A4 = 440 Hz, A5 = 880 Hz
 * 
 * PIEZO BUZZER TYPES:
 * - Active: Contains oscillator, makes tone when powered
 * - Passive: Needs frequency input, use with tone()
 * 
 * This project uses a PASSIVE buzzer!
 */

5. Explanation & Challenges

üîç How It Works

  1. tone(pin, freq, duration): Generates square wave at specified frequency.
  2. Frequencies: Each musical note has a specific frequency (A4 = 440 Hz).
  3. Duration: Quarter notes, half notes, etc. are calculated from tempo.
  4. Arrays: Melody and durations stored in parallel arrays.
