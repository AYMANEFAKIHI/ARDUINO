Project #81: NTP Clock (Internet Time)

Get the precise time from the internet using Network Time Protocol (NTP).

1. Concept and Description

No need to set the time manually. Your ESP8266 can fetch the exact time from NTP servers.

2. Components Needed

  • ESP8266 (NodeMCU)

3. Physical Montage (Wiring Instructions)

None.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.

#include 
#include 
#include 

const char *ssid     = "SSID";
const char *password = "PASS";

WiFiUDP ntpUDP;
NTPClient timeClient(ntpUDP, "pool.ntp.org");

void setup(){
  Serial.begin(115200);
  WiFi.begin(ssid, password);
  timeClient.begin();
  timeClient.setTimeOffset(3600); // GMT+1
}

void loop() {
  timeClient.update();
  Serial.println(timeClient.getFormattedTime());
  delay(1000);
}
