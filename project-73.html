Project #73: MicroSD Card Data Logger

Save sensor data to a text file on a MicroSD card.

1. Concept and Description

Data logging is essential for long-term monitoring. We'll write data to a CSV file on an SD card.

2. Components Needed

  • Arduino Uno
  • MicroSD Card Module
  • SD Card

3. Physical Montage (Wiring Instructions)

CS to D10, MOSI to D11, MISO to D12, SCK to D13. VCC to 5V.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.


#include 
#include 

const int chipSelect = 10;

void setup() {
  Serial.begin(9600);
  if (!SD.begin(chipSelect)) {
    Serial.println("Card failed, or not present");
    return;
  }
}

void loop() {
  File dataFile = SD.open("datalog.txt", FILE_WRITE);
  if (dataFile) {
    dataFile.println("Sensor Data Here");
    dataFile.close();
    Serial.println("Saved.");
  }
  delay(1000);
}
