Project #71: PCA9685 PWM Driver (16 Servos)

Control up to 16 servos using just 2 pins (I2C) with the PCA9685 driver.

1. Concept and Description

The Arduino has limited PWM pins. This driver allows you to control many servos, perfect for robot arms or hexapods.

2. Components Needed

  • Arduino Uno
  • PCA9685 Module
  • Servos
  • External 5V Power

3. Physical Montage (Wiring Instructions)

VCC to 5V, GND to GND, SDA to A4, SCL to A5. External power to terminal block.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.


#include 
#include 

Adafruit_PWMServoDriver pwm = Adafruit_PWMServoDriver();

void setup() {
  pwm.begin();
  pwm.setPWMFreq(60);
}

void loop() {
  // Drive servo on channel 0
  for (uint16_t pulselen = 150; pulselen < 600; pulselen++) {
    pwm.setPWM(0, 0, pulselen);
  }
  delay(500);
  for (uint16_t pulselen = 600; pulselen > 150; pulselen--) {
    pwm.setPWM(0, 0, pulselen);
  }
  delay(500);
}
