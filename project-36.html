Project #36: NeoPixel Rainbow Cycle

In this project, we will create a smooth, flowing rainbow animation on our NeoPixel strip. This effect is one of the most popular uses for addressable LEDs and demonstrates how to use loops and color math to create dynamic patterns.

1. Concept and Description

In this project, we will create a smooth, flowing rainbow animation on our NeoPixel strip. This effect is one of the most popular uses for addressable LEDs and demonstrates how to use loops and color math to create dynamic patterns.

2. Components Needed

  • Arduino Uno Board
  • USB Cable
  • NeoPixel LED Strip (or Ring/Stick)
  • Jumper Wires

3. Physical Montage (Wiring Instructions)

The circuit is identical to the previous project:

  • 5V to Arduino 5V
  • GND to Arduino GND
  • DIN (Data In) to Arduino Pin 6

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.


#include <Adafruit_NeoPixel.h>

#define PIN        6
#define NUMPIXELS  8 

Adafruit_NeoPixel pixels(NUMPIXELS, PIN, NEO_GRB + NEO_KHZ800);

void setup() {
  pixels.begin();
  pixels.setBrightness(50);
}

void loop() {
  // Cycle through the rainbow colors
  // The first number (10) is the wait time between updates (speed)
  rainbow(10);
}

// Custom function to create a rainbow cycle
void rainbow(int wait) {
  // Hue of first pixel runs 5 complete loops through the color wheel.
  // Color wheel has a range of 65536 but it's OK if we roll over.
  for(long firstPixelHue = 0; firstPixelHue < 5*65536; firstPixelHue += 256) {
    
    for(int i=0; i<pixels.numPixels(); i++) { 
      // For each pixel in strip...
      // Offset pixel hue by an amount to make one full revolution of the
      // color wheel (range of 65536) along the length of the strip
      // (pixels.numPixels() steps):
      int pixelHue = firstPixelHue + (i * 65536L / pixels.numPixels());
      
      // pixels.ColorHSV() converts hue to an RGB color
      pixels.setPixelColor(i, pixels.gamma32(pixels.ColorHSV(pixelHue)));
    }
    pixels.show(); // Update strip with new contents
    delay(wait);  // Pause for a moment
  }
}
