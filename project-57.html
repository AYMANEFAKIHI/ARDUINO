Project #57: IR Remote Control Decoder

Read the codes sent by your TV or DVD remote control.

1. Concept and Description

Infrared (IR) remotes send unique codes for each button. This project decodes and displays them.

2. Components Needed

  • Arduino Uno
  • IR Receiver (VS1838B)
  • Remote Control

3. Physical Montage (Wiring Instructions)

Receiver Out to D11, VCC to 5V, GND to GND.

4. The Code (Arduino Sketch)

Copy and paste the code below into your Arduino IDE.


#include 

const int RECV_PIN = 11;
IRrecv irrecv(RECV_PIN);
decode_results results;

void setup(){
  Serial.begin(9600);
  irrecv.enableIRIn();
}

void loop(){
  if (irrecv.decode(&results)){
    Serial.println(results.value, HEX);
    irrecv.resume();
  }
}
